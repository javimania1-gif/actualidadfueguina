# Politica de fuentes

## Tier A
Fuentes primarias autoritativas: organismos gubernamentales, municipios, justicia competente, legislaturas, autoridades electorales, organismos deportivos oficiales, servicios meteorologicos oficiales y organismos cientificos para datos propios.

Una Tier A puede bastar solo dentro de su competencia.

## Tier B
Medios periodisticos directos. Para high-risk, una sola Tier B queda pendiente; dos publishers independientes y concordantes pueden verificar.

## Tier C
Agregadores como Google News y Bing News. Sirven para descubrir, no para corroborar por si mismos.

Google + Bing apuntando al mismo publisher cuentan como una sola fuente editorial.

## URL valida
No son fuente valida: homepages, portadas, buscadores, categorias genericas, paginas de login/error o textos insuficientes. La validacion vive en `scripts/lib/source-policy.mjs`.
